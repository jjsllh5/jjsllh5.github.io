## 1.게임내용
### 게임이름: Aim Shot
### 게임플랫폼: 윈도우 데스크톱
### 게임특징:제한시간안에 적을 맞추면서 점수가 올라가는 형식이면서 그 점수로 탄약, 회복을 보급하는 형식 
### 장르: FPS
### 게임 조작법
#### 이동: 앞W, 뒤S, 왼쪽A, 오른쪽D
#### 점프: 스페이스바
#### 뛰기: Shift
#### 무기: 1번 망치, 2번 권총, 3번 서브머신건
#### 재장전: R
#### 공격: 마우스 좌클릭

### 4주차: 무기와 일반 적 구현하기
### 5주차: 체력,점수,탄약,시간 표시,총사운드
### 6주차: 적 피격, 플레이어 피격 구현
### 7주차: 원거리적 구현
### 8주차: 중간고사
### 9~10주차: 돌진하는적 구현, 총구 화염 제작
### 11주차: 상점 구현
### 12주차: 스테이지, 총사운드(문제해결), 건물 구현
### 13주차: 스테이지마다 몬스터생성, 보스몬스터 구현
