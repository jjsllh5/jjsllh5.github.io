## 1.게임내용
### 게임이름: Aim Shot
### 게임플랫폼: 윈도우 데스크톱
### 게임특징:제한시간안에 적을 맞추면서 점수가 올라가는 형식이면서 그 점수로 탄약, 회복을 보급하는 형식 
### 장르: FPS
### 게임 조작법
#### 이동: 앞W, 뒤S, 왼쪽A, 오른쪽D
#### 점프: 스페이스바
#### 뛰기: Shift
#### 무기: 1번 망치, 2번 권총, 3번 서브머신건
#### 재장전: R
#### 공격: 마우스 좌클릭
